# Fatigue and Damage-Tolerance Screening

Planned sequence:

1. freeze stress extraction location and notch method;
2. define representative spectrum with provenance;
3. implement rainflow/bin counting where needed;
4. mean-stress correction and S-N screening;
5. Miner damage with scatter sensitivity;
6. initial flaw assumptions and residual-strength limit;
7. Paris/NASGRO/AFGROW crack-growth screening;
8. inspection-interval sensitivity.

No result is to be described as certification substantiation. Material crack-growth data and fracture toughness must be tied to product form/environment and kept separate from nominal tensile properties.
